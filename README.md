## ğŸš€ Installation

### For Users

1. Download the latest release from the [Chrome Web Store](https://chrome.google.com/webstore) (Coming Soon).
2. Pin the extension to your browser toolbar.
3. Log in to your NIBM LMS account to start syncing.

### For Developers

1. Clone the repository:
```bash
git clone https://github.com/your-username/lms-activity-manager.git

```


2. Install dependencies:
```bash
npm install

```


3. Build the extension:
```bash
npm run build

```


4. Load into Chrome:
* Open `chrome://extensions/`.
* Enable **Developer mode**.
* Click **Load unpacked** and select the `dist` folder.
